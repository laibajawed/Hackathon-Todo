# Todo API with AI Chatbot

FastAPI backend for Todo application with AI chatbot functionality.

## Features
- JWT Authentication
- PostgreSQL Database (Neon)
- AI Chatbot with Groq LLM
- MCP Server Integration
- RESTful API

## Environment Variables Required
- DATABASE_URL
- JWT_SECRET
- GROQ_API_KEY
- CORS_ORIGINS
- ENVIRONMENT=production
